About the Hackathon

The TECHNOVA Web Development Hackathon 2025 is a two-day technical event designed to give students hands-on experience in building real-world web projects using front-end technologies.

The event focuses on creativity, problem-solving, and practical implementation of ideas through modern and interactive websites.

Objectives

  • Provide hands-on experience in front-end web development
  • Encourage creativity and innovative thinking
  • Enhance teamwork, confidence, and technical skills

Theme & Project Ideas

Theme: Creative Web Solutions Using Front-End Technologies

Teams are encouraged to design creative and meaningful websites that solve real-life problems or present innovative ideas.

Event Schedule

  • 22 December 2025: Coding & Development Phase
  • 23 December 2025: Project PPT Presentation

During the presentation round, teams must explain their project idea, technologies used, features, screenshots, challenges faced, solutions, and future enhancements.

Rules & Regulations

  • Each team must consist of 2 members only
  • One member must be designated as the Team Leader
  • Projects must be developed using HTML, CSS, and JavaScript only
  • Project must be original and completed during the event time
  • Final code submission is mandatory before the deadline
  • Projects must be uploaded to GitHub
  • No editing is allowed after submission

Participation

Team Size: 2 Members

Eligibility: All Courses
